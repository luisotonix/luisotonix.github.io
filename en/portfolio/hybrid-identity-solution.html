The Challenge

The company operated with multiple Active Directory forests in different countries, resulting in identity silos, fragmented user experience and security risks. Employees needed to remember multiple passwords and there was no centralized visibility over access and suspicious activities.

Additionally, SaaS application adoption was growing rapidly, but without an SSO system, each new application meant one more credential to manage. The help desk was overwhelmed with password reset tickets.

The Solution

I designed and implemented a unified hybrid identity solution using Microsoft Entra ID (Azure AD) as the central identity hub, maintaining integration with on-premises Active Directory.

Phase 1: Assessment and Design

  • Mapping of all AD forests and domains
  • Application inventory and authentication methods
  • Synchronization architecture design
  • UPN suffix strategy definition

Phase 2: Azure AD Connect

  • High availability Azure AD Connect implementation
  • Password Hash Sync + Seamless SSO configuration
  • Synchronization filters by OU and group
  • Azure AD Connect Health for monitoring

Phase 3: Identity Security

  • MFA rollout for all users
  • Risk-based Conditional Access policies
  • Identity Protection for threat detection
  • Privileged Identity Management for admins

Phase 4: Single Sign-On

  • Integration of 150+ SaaS applications with SAML/OIDC
  • Enterprise Applications configuration
  • My Apps portal for users
  • Automatic provisioning (SCIM) for critical apps

Phase 5: Passwordless

  • Windows Hello for Business
  • Microsoft Authenticator app
  • FIDO2 security keys for admins
  • Self-Service Password Reset (SSPR)
